# Py4GW

# This repository is retired

Py4GW has been rebuilt from the ground up and now lives at:

    https://github.com/apoguita/Py4GW_Reforged

Please use the new repository for everything going forward:

  - Latest code and updates
  - Downloads and releases
  - Issues, questions, and pull requests

The code in this repository is frozen at its final state. It is kept online
only as a historical archive and will no longer receive updates, fixes, or
support. Anything you were using here can now be found in Py4GW_Reforged.

Thank you to everyone who used and contributed to the original Py4GW.
